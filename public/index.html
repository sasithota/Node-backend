User Management system

Available routes

/api/register

PUT

user can create account by inserting username and password in request body

expect msg containing username and error

/api/login

POST

user can login to his/her account by inserting username and password in request body

expect msg containing username and authentication token and error

/api/posts

PUT

user can create his/her posts by inserting autentication token in request header and post's title and content in requrest body

expect msg containing postand error

/api/posts

GET

user can view his/her posts by inserting authentication in request header

expect msg containing username and error

/api/posts/:postid

POST

user can update a post by inserting post's title and content in request body and authentication token in requrest header

expect msg containing username and error
